# JavaScript Fundamental

## Number

```javascript
const num1 = 100;
const num2 = 50;
let val;

// Simple math with numbers
val = num1 + num2;
val = num1 * num2;
val = num1 - num2;
val = num1 / num2;
val = num1 % num2;

// Math Object
val = Math.PI;
val = Math.E;
val = Math.round(2.4); // result 2
val = Math.ceil(2.4); // result 3
val = Math.floor(2.8); // result 2
val = Math.sqrt(64); // result 8 (akar)
val = Math.abs(-3); // result 3 (absolut number)
val = Math.pow(8,2); // result 64 (power atau pangkat)
val = Math.max(2,3,2,6,7,9,-1); // result -9
val = Math.min(2,3,2,6,7,9,-1);

val = Math.random();
val = Math.random() * 20;
val = Math.random() * 20 + 1;
val = Math.floor(Math.random() * 20 + 1);







console.log(val);
```
